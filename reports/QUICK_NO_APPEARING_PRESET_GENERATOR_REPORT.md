# Quick No. Appearing Preset Generator Report

Generated 926 monster presets from the current embedded Quick No. Appearing preset data.

The preset picker uses a searchable browser datalist instead of a very large select menu. `None` is the first preset and clears the picker back to manual control.

Preset choices are heuristic referee-facing best guesses. They use monster name, HD/HDE, source description keywords, and explicit overrides for common edge cases such as dragons, golems, oozes, undead hordes, pack predators, herd animals, rare outsiders, site-bound hazards, and social humanoids.
